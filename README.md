# Advanced-CNN-model

Biocon Ltd. is working on automated diagnosis of blood samples. Develop an advanced CNN model to classify red blood 
cell images as healthy or malaria-infected, addressing class imbalance, image noise, and variability, while incorporating 
model interpretability (e.g., Grad-CAM) to assist medical experts in validating predictions.
